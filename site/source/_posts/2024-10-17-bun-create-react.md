# 使用bun创建一个react应用程序

## 想在你的下一个 React 项目中使用 Bun 吗？

### 1. 安装`bun`

首先运行以下命令检查`bun`是否安装

```shell
bun -v
```

如果尚未`bun`安装则运行以下命令进行安装`bun`

```shell
curl https://bun.sh/install | bash
```
